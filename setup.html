⚙️ 乒乓之家 · 后端配置

填入你的 Supabase 项目信息，自动写入所有页面。只需配置一次。

第一步：填入 Supabase 信息

在 app.supabase.com → 你的项目 → Settings → API

第二步：点击配置

✅ 配置写入成功！

现在打开 pingpong_home.html 即可使用真实后端。

如果是第一次使用，还需要在 Supabase SQL Editor 执行 supabase_setup.sql
💡 找不到 anon key？
登录 Supabase → 选择项目 → 左侧菜单 Settings → API → 找到 anon public 那行 → 点 Copy

第三步：执行数据库建表脚本

在 Supabase 控制台 → SQL Editor → 粘贴 supabase_setup.sql 全部内容 → Run

只需执行一次，之后数据会自动管理。
